## 技术栈

| 层 | 技术 | 说明 |
|---|---|---|
| 后端 | Spring Boot 3.2.1 / Java 17 | 主框架 |
| ORM | MyBatis-Plus 3.5.5 | 数据库操作 |
| 数据库 | MySQL 8 | localhost:3306 / private_aggregator |
| 缓存 | Redis + Caffeine | 二级缓存 |
| 消息队列 | RabbitMQ | 异步任务 |
| 鉴权 | Sa-Token 1.38 | Token 认证 |
| 分布式锁 | Redisson | 并发控制 |
| 对象存储 | MinIO / 阿里云 OSS / 本地磁盘 | 存储策略可切换 |
| 前端 | Vue 3 + Vite + Element Plus + Pinia | SPA |
| 文档 | Knife4j (Swagger) | API 文档 |
| 部署 | Docker Compose | 一键部署 |
