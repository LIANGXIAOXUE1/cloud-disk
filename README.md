# Cloud Disk — 私人聚合云盘功能介绍与使用场景

> 版本：v1.0 | 2026-06-02

---

## 一、项目一句话介绍

**Cloud Disk** 是一款自部署的私人聚合云盘，将你自己的电脑变成一台个人文件中心。统一管理本地文件与百度网盘文件，在任何设备上通过浏览器访问，享受文件上传、预览、分享、AI 笔记等一站式体验。数据完全在你自己的硬盘上，不经过任何第三方服务器。

---

## 二、核心功能一览

### 2.1 文件管理）

| 功能           | 说明                                 |
| ------------ | ---------------------------------- |
| **文件夹管理**    | 创建文件夹、层级导航、面包屑路径，像操作本地文件管理器一样简单    |
| **文件上传**     | 支持拖拽上传 / 点击上传，小文件直传，大文件自动分片 + 断点续传 |
| **文件搜索**     | 输入关键词即时搜索文件，300ms 防抖               |
| **重命名 / 移动** | 随时重命名文件，移动到其他文件夹                   |
| **分享链接**     | 一键生成分享链接，可设提取密码，支持取消分享             |
| **回收站**      | 删除的文件先移入回收站，可恢复或彻底删除               |

### 2.2 存储策略

| 策略         | 说明                                                       |
| ---------- | -------------------------------------------------------- |
| **本地磁盘存储** | 文件直接存储在 `D:\cloud-disk-files\`，零外部依赖                     |
| **策略模式设计** | 预留 `ILocalStorageService` 接口，未来切换 MinIO / 阿里云 OSS 只需加实现类 |
| **目录结构**   | upload（普通上传）/ temp（分片临时）/ baidu（百度同步）三个子目录               |

### 2.3 在线预览

这是本次新增的核心功能模块，让云盘从"文件仓库"升级为"内容中心"。

| 文件类型                   | 预览方式          | 亮点                                        |
| ---------------------- | ------------- | ----------------------------------------- |
| **图片**                 | 灯箱预览器         | 放大/缩小/旋转/左右切换，支持 jpg/png/gif/webp/svg/bmp |
| **PDF**                | pdf.js 在线渲染   | 翻页、缩放、目录跳转、文本搜索                           |
| **Word / Excel / PPT** | OnlyOffice 渲染 | 直接在浏览器里查看 Office 文档，无需下载                  |
| **视频**                 | video.js 播放器  | **倍速播放**（0.5x~2x）、进度拖拽、画中画、快捷键            |
| **音频**                 | 内嵌音频播放器       | 播放列表、进度拖拽、音量控制                            |
| **文本 / 代码**            | 语法高亮展示        | .txt/.md/.json/.py/.java 等文件在线高亮阅读        |

### 2.4 AI 智能笔记

| 功能           | 说明                                  |
| ------------ | ----------------------------------- |
| **文档 AI 摘要** | 上传 PDF/Word 文档后，一键生成内容摘要和关键要点       |
| **视频 AI 笔记** | 看视频时 AI 自动生成文字摘要 + 分段要点 + 时间轴标注     |
| **图片 AI 描述** | 查看图片时 AI 自动描述图片内容，提取图中文字（OCR）       |
| **笔记管理**     | AI 生成的笔记自动保存为 Markdown 文件，可查看/编辑/导出 |

### 2.5 百度网盘同步（待开发）

| 功能       | 说明                      |
| -------- | ----------------------- |
| **一键授权** | 授权自己的百度网盘账号             |
| **自动同步** | 百度网盘文件自动下载到本地磁盘指定目录     |
| **统一管理** | 百度网盘文件与本地文件在同一页面展示，无缝切换 |

## 三、使用场景

### 场景 1：学生 / 研究者的资料管理

> 小张是一名研究生，每天需要阅读大量 PDF 论文、整理 Word 笔记、收集实验视频。他用 Cloud Disk：

- 把论文 PDF 上传到「论文」文件夹，在线预览不用打开本地阅读器
- 看重要的论文时，点击"AI 摘要"一键获取核心观点，不用逐页读
- 录制的实验视频上传后在线播放，倍速回看关键片段，AI 自动生成实验要点笔记
- 百度网盘收藏的公开课视频自动同步到本地，统一管理

### 场景 2：上班族的文件中心

> 小王是产品经理，PRD 文档、竞品截图、会议录音散落在各个角落。他用 Cloud Disk：

- 把 Word PRD 文档上传后，发给同事一个分享链接 + 密码，对方在线预览无需下载
- 竞品截图上传后灯箱预览，左右切换对比，AI 自动描述图片内容
- 会议录音上传后在线播放，AI 自动生成会议纪要

### 场景 3：自由职业者的作品集

> 小陈是视频剪辑师，经常需要给客户展示作品、收集素材。他用 Cloud Disk：

- 把剪辑好的视频上传，生成分享链接发给客户，客户在线预览 + 倍速查看
- 项目素材自动分片上传，网络断了也不怕，下次续传
- 客户反馈的 PDF 标注文件上传后直接在线看，不用下载 Adobe Reader

### 场景 4：个人数据私有化

> 不想把个人文件放在公有云上，但又需要随时随地访问。

- 所有文件 100% 存在自己的电脑硬盘上，不经任何第三方服务器
- 局域网内手机/平板打开浏览器就能访问，上传下载都在内网极速传输
- 百度网盘文件一键同步到本地，从此不怕网盘限速

## 四、技术架构

```
┌─────────────────────────────────────────────┐
│                  前端 (Vue 3)                │
│  登录 → 文件管理 → 在线预览 → AI 笔记         │
│  图片灯箱 / PDF.js / video.js / OnlyOffice   │
└─────────────────┬───────────────────────────┘
                  │ HTTP (Axios + Sa-Token)
┌─────────────────▼───────────────────────────┐
│              后端 (Spring Boot)              │
│  用户认证 → 文件 CRUD → 分享 → 回收站        │
│  上传管理 → AI 摘要服务 → 百度网盘同步        │
└─────────────────┬───────────────────────────┘
                  │
┌─────────────────▼───────────────────────────┐
│             存储层（策略模式）                 │
│  本地磁盘 (当前)  │  MinIO  │  阿里云 OSS    │
│  D:\cloud-disk-files\                       │
└─────────────────────────────────────────────┘
```
